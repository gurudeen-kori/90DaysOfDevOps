
# 90DaysOfDevOps

## Process Management
- List processes: `ps aux`
- Monitor: `top` or `htop`
- Kill: `kill PID` or `pkill name`
- Priority: `nice/renice`
- Services: `systemctl status/start/stop service`
# -----------------------------------------------------------------------------
## File System
- Mount: `mount/umount`
- Check: `fsck`
- Space: `df -h`, `du -sh`
- Permissions: `chmod`, `chown`
- Types:
   `1. ext4`, 
   `2. XFS`, 
`  3. Btrfs`
# -----------------------------------------------------------------------------
## Networking Troubleshooting
- IP config: `ip addr`
- Hosts config: `/etc/hosts`
- Ping: `ping host`
- Trace: `traceroute host`
- DNS: `nslookup/dig`
- Sockets: `ss -tuln`
- ssh connection : `remote connection `
 `1. key generation and copyed to the remode host through ip address `
 `2.ssh <username@ip> to connect the remote host  `
